¿Qué es entonces una base de datos - DB?

Es una colección organizada de datos almacenada digitalmente en un sistema de computadoras.
-https://www.oracle.com/mx/database/what-is-database.html



¿En qué casos es conveniente usar bases de datos?

Cuando se necesita acceder a un segmento de información de manera repetida y organizada.



¿Qué es un sistema de gestión de base de datos - DBMS?

Servir como interfaz entre el usuario y la base de datos.
-Avi Silberschatz, Henry F. Korth, S. Sudarshan, Database System Concepts, pag 50-126



Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Abstracción de datos
Seguridad de datos
Manejo eficiente de datos
Simplicidad de acceso
